## Development

```bash
git clone https://github.com/xcpcio/xcpcio.git

# Installation dependencies
pnpm install

# build the project
pnpm build

# Start docs development server
pnpm docs

# Start board development server
pnpm board

# Run tests
pnpm test

# Lint all code
pnpm lint

# Fix lint issues automatically
pnpm lint:fix
```
